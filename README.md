# Shareable OpenClaw Pack

一个可分享的 OpenClaw 配置模板仓库，包含：
- 主 agent 模板
- 可复用的工作风格与编排规则
- skills 目录占位

## 内容
- `agents/main/AGENTS.md`：主控 agent 编排规则模板
- `agents/main/SOUL.md`：agent 风格模板
- `agents/main/TOOLS.md`：本地工具说明模板
- `skills/`：用于放可分享 skill

## 注意
这个仓库已经做了脱敏处理：
- 不包含真实用户信息
- 不包含记忆文件
- 不包含密钥、密码、域名私配
- 不包含 `.env`

## 使用方式
把需要的文件复制到你的 OpenClaw workspace，或作为仓库直接维护。
