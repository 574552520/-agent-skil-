# Environment Variables

Use private env storage; do not commit secrets.

## Required for image generation
- IMAGE_API_KEY
- IMAGE_API_BASE_URL

## Required for video generation
- VIDEO_API_KEY
- VIDEO_API_BASE_URL

## Usually needed for returning results
- PUBLIC_ASSET_BASE_URL
- STORAGE_BUCKET
- RESULT_CALLBACK_BASE_URL

## Optional operational variables
- IMAGE_MODEL
- VIDEO_MODEL
- IMAGE_TIMEOUT_SECONDS
- VIDEO_TIMEOUT_SECONDS
